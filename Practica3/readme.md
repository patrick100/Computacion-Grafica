Instrucciones
Para Pintar
W  Mover arriba <br/>
A  Mover izquierda <br/>
S  Mover abajo <br/>
D  Mover derecha <br/>

I  Rotar arriba <br/>
J  Rotar izquierda <br/>
K  Rotar abajo <br/>
L  Rotar derecha <br/>

'+'  Aumentar escala <br/>
'-'  Disminuir escala <br/>
